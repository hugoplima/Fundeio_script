📦 Fundeio_script

Scripts MATLAB para processamento de dados de fundeio (análise hidrográfica).

Este repositório contém um conjunto de scripts em MATLAB que auxiliam no processamento e análise de dados coletados durante operações de fundeio (medidas hidrográficas e similares), junto com um manual de instruções.

📌 Conteúdo

Os principais arquivos e diretórios são:

MANUAL DOS SCRIPTS.docx – Manual com instruções sobre os scripts.
subrotinas/ – Sub-rotinas auxiliares usadas pelos scripts principais.
.m (arquivos MATLAB) – Scripts para processamento de dados, como:
adp_matriz.m
hidro_NOVA2.m
isopleta_mare_NOVA.m
…e outros scripts relacionados ao processamento de perfil e interpolação.
🛠️ Requisitos

Para executar os scripts você precisa de:

MATLAB (versão compatível com os scripts).
Pacotes/toolboxes necessários (dependendo de chamadas específicas dos scripts).
🚀 Como usar
Coloque todos os arquivos .m e a pasta subrotinas no mesmo diretório de trabalho do MATLAB.
Abra o MATLAB e navegue até esse diretório.
Execute os scripts desejados diretamente no MATLAB, por exemplo:
>> adp_matriz
Consulte o MANUAL DOS SCRIPTS.docx para detalhes de entrada, parâmetros e exemplos.
📝 Detalhes dos Scripts

Cada script faz parte de um fluxo de processamento. Por exemplo:

adp_matriz.m – possível processamento de dados ADP.
hidro_*.m – scripts relacionados a dados hidrográficos.
isopleta_*.m – gera curvas de isopletas (contornos) a partir dos dados.

(O significado específico de cada script deve ser consultado no manual do projeto.)
